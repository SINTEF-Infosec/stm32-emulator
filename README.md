STM32 Emulator
==============

This is a work in progress. The end-goal is to simulate 3D printers.

Example usage
--------------

```
$ cd example/
$ cargo run --release -- config.yaml --stop-addr 0x080201f8 -v --color=always | grep -vE 'PB(8|9)'
```
